  CGPA Calculator (React App)

This is a simple and responsive "CGPA Calculator" built using React.js 
It allows users to enter their "GPA" and "Credit Hours" for multiple subjects and calculates the overall CGPA using a weighted average formula.


  Features
  
- Input GPA and Credit Hours for 3 subjects  
- Calculates accurate CGPA based on weighted average  
- Clean, modern, and responsive UI  
- Built as an internship project to practice React fundamentals  



 Technologies Used

- React.js  
- HTML5  
- CSS3  
- JavaScript (ES6)
